## Development

### Initialize virtual enviroment (optional)

```sh
python -m venv .venv
```
Windows (cmd.exe)
```sh
.venv/Scripts/activate.bat
```
Windows (PowerShell)
```sh
.venv/Scripts/Activate.ps1
```
Linux/MacOS
```sh
source .venv/bin/activate
```


### Install dependencies

```sh
pip install -r requirements.txt
```
